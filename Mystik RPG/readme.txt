2. Controls

Controls:
Up Arrow key: moves Up
Right Arrow key: moves Right
Down Arrow key: moves Down
Left Arrow key: moves Left
Z key: picks up objects under you
X key (on Inv.): drops item
LSHIFT: views signs
CTRL key: toggles stats in-game. SHIFT+ARROW key: open doors
